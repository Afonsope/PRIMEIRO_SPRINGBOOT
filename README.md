# 🚀 Projeto - PRIMEIRO_SPRINGBOOT

## 📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de criar uma **aplicação backend utilizando Spring Boot**, explorando os primeiros conceitos do framework e a construção de APIs REST.

A aplicação segue boas práticas de desenvolvimento backend, utilizando uma estrutura organizada em camadas, padrão amplamente utilizado em projetos Java com Spring Boot ([GitHub][1]).

---

## 🧠 Funcionalidades

✅ Criação de endpoints REST
✅ Estruturação em camadas (Controller, Service, Repository)
✅ Manipulação de dados no backend
✅ Integração com banco de dados

---

## 🛠️ Tecnologias Utilizadas

**Backend:** Java + Spring Boot
**Banco de Dados:** MySQL (ou outro, conforme configuração)
**Build Tool:** Maven
**Outros:** API REST, Arquitetura em Camadas

---

## 📂 Estrutura do Projeto

```
/src
  /controller
  /service
  /repository
  /model
/resources
pom.xml
```

---

## ⚙️ Como Executar o Projeto

1. Clone ou baixe este repositório
2. Abra o projeto em uma IDE (IntelliJ, Eclipse ou VS Code)
3. Configure o banco de dados no arquivo `application.properties`
4. Execute a aplicação (classe principal com Spring Boot)
5. Acesse os endpoints via navegador ou ferramenta como Postman

---

## 🎯 Objetivo

Este projeto foi desenvolvido para fins de aprendizado e prática, aplicando conceitos como:

* Desenvolvimento de APIs REST
* Arquitetura backend com Spring Boot
* Organização de código em camadas
* Integração com banco de dados
* Fundamentos de aplicações Java modernas
